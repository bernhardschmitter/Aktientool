Aktientool V3.17 Workflow-Fix

Diese ZIP enthält die beiden wichtigen Dateien im korrekten Ordner:

.github/workflows/update-prices.yml
.github/scripts/update-prices.js

Wichtig:
Der Ordner .github ist auf iPad/Mac eventuell unsichtbar, weil er mit einem Punkt beginnt.
Falls der Upload über das iPad nicht klappt, bitte die Dateien in GitHub manuell anlegen:

1) Neue Datei anlegen mit Namen:
.github/workflows/update-prices.yml

2) Neue Datei anlegen mit Namen:
.github/scripts/update-prices.js

Danach unter Actions prüfen, ob "Update Prices" erscheint.
